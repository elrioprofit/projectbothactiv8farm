# AgroBot — Project HACTIV8

## Project
AgroBot adalah aplikasi chatbot fullstack berbasis Gemini AI — konsultan pertanian, perlindungan tanaman, dan agronomi kelapa sawit.

## Quick Start
```bash
npm install
cp .env.example .env  # isi GEMINI_API_KEY
npm start             # http://localhost:3000
```

## Tech Stack
Express.js + @google/genai + Vanilla HTML/CSS/JS (Glassmorphism)

## Key Files
- `index.js` — Backend server
- `bahan frontend/starter/` — Frontend (index.html, style.css, script.js)
- `.env` — API key (JANGAN di-commit)
- `tasks/project-context.md` — Context document
